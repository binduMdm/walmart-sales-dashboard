# 📊 Walmart Weekly Sales Dashboard

This Power BI dashboard analyzes Walmart’s weekly sales data to uncover how seasonal trends, holidays, and temperature impact sales across various store locations.

---

## 🎯 Project Goal

The goal of this project was to explore how weekly sales are influenced by holidays and weather changes, and identify top-performing stores across different months.

---

## 🛠️ Tools & Technologies Used

- **Power BI** – for creating interactive dashboards and slicers
- **Power Query** – for data cleaning and transformation
- **DAX** – for calculated fields and KPIs
- **Bookmarks & Selection Pane** – for interactive filter panels
- **CSV (Walmart dataset)** – as the data source

---

## 📈 Key Features

- Holiday vs Non-Holiday Sales Comparison  
- Store-wise Sales Performance Visualization  
- Dual-axis chart showing Temperature vs Sales trends  
- Interactive Filter Panel using bookmarks and toggle buttons  
- Dynamic tooltips and slicers for user exploration

---

## 📊 Key Insights

- Sales tend to rise in warmer months and peak around July and December.
- Stores 5, 12, and 20 consistently outperform others during holiday weeks.
- There is a seasonal pattern that aligns with temperature increases.
- Toggleable filter buttons allow users to customize their view without crowding the interface.
